Summary

This project is designed to demonstrate object oriented programming with Python, while also demonstrating MVC design patterns. The project will demonstrate major differences between Python and other languages, such as the need (or lack thereof) for Getters/Setters.

Students will be able to write a TKinter Interface and connect it with functions and values from an object. Students will also understand the object hierarchy and how to reference objects in built-in and custom classes. Students will be exposed to multi-threaded programming and some of the Python libraries available for threading.

Pet Object - Model

The pet object is just an object representation of a fictitious pet.

  • Pet
    • Hunger

    • Hunger Rate
    • Happiness Rate
    • Cleanliness Rate

Each value held in the Pet object model represents a characteristic of the pet.

Pet Interface - View

The interface will feature an image, that changes with each action performed, a button for each action, a label, and a value for each characteristic of the pet.
